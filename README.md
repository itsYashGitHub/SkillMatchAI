# SkillMatch AI

AI-powered Resume Screening & Job Matching System

## Overview

This project uses natural language processing (NLP) and machine learning (ML) techniques to calculate how well a candidate's resume matches a given job description.

## Features

- Calculates match percentage using TF-IDF and cosine similarity
- Simple web interface built with Streamlit
- Beginner-friendly codebase ready to extend

## How to run

1. Clone the repo  
2. Create and activate a virtual environment  
3. Install dependencies: `pip install -r requirements.txt`  
4. Run the app: `streamlit run app.py`
